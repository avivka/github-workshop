# GitHub Enterprise Administrator Workshops

This repository contains a series of hands-on workshops designed for GitHub Enterprise and Organization Administrators.

## Workshop Modules

Each module focuses on a specific aspect of enterprise governance and security.

1. [Workshop 01: Managing GitHub Rulesets](workshops/01-rulesets.md)
   - Learn how to enforce granular branch and repository policies using `fnmatch` filters.
   - Topics: Production protection, QA isolation, and mandatory PR workflows.

2. [Workshop 02: Enterprise Actions Management](workshops/02-enterprise-actions.md)
   - Scale and standardize automation using reusable workflows and composite actions.
   - Topics: Standardization, policy enforcement, and admin-level governance.

3. [Workshop 03: Enterprise Security Strategy](workshops/03-security-policies.md)
   - Protect the supply chain and enable security baselines.
   - Topics: Actions Allow Lists, Secret Scanning, Dependabot.

4. [Workshop 04: Audit & Compliance](workshops/04-audit-compliance.md)
   - Investigate security incidents using Audit Logs.
   - Topics: Incident investigation, Search syntax, Log streaming.

5. [Workshop 05: Terraform & Self-Service](workshops/05-iac-self-service.md)
   - Automate repository creation using Templates and IaC.
   - Topics: Terraform provider, Self-service automation.

---

## Getting Started

1. Clone this repository.
2. Explore the file structure (`app/`, `prod/`, `qa/`).
3. Follow the instructions in the specific workshop module links above.
